Get Involved

2019-11-29

There are several ways that you can get involved with the Xenon project. Here are some suggestions:

  • Submit a bug report
  • Create a Mod
  • Create a pull request

Submitting Bug Reports

Since the project is hosted on GitHub, we use the GitHub issue tracking. When submitting a bug please give as much information about the problem as possible. Screenshots or videos are also helpful when demonstrating a problem. Also report what version of Xenon you are using and what mods you have installed (both enabled and disabled).

Create a Mod

Xenon depends on mods from people like you who can provide creative tools and features. If you already program in Java, building a mod is pretty easy. If you are not a programmer, this just might be the time to get started. Check out the mod development instructions for further information.

Create a Pull Request

If you familiar with Java development please consider forking the Xenon project and providing pull requests for bug fixes and new features. If you would like to coordinate with the project team please contact them through the Xenon GitHub team.
