Tool Guide

2024-08-25

Actions are the basic user input activity in Xenon. Actions are used to perform tasks such as opening a file, saving a file, or printing a file. Actions are typically associated with a menu item or a toolbar button.

There are several steps to creating an action in Xenon:

  1. Add the action data to the action resource bundles.
    • You may need to create a new icon for your new action
    • Be sure to register your new icon in the icon library
    • Be sure your translations are clear in other languages
  2. Register the action with the program action manager.
  3. Create an action class that extends Action.
    • Implement the isEnabled() method.
    • Implement the execute() method.
  4. Push and pull the action in the tool.
