Mod Development

Project Setup

To start a new mod project it is recommended that you use Maven, or Maven compatible tools like Gradle, to configure the new mod project using an existing parent pom to jump start the process. It is possible to set up a mod project without using the parent pom but you would have to handle the configuration on your own.
